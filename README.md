# UnoJoystick
UnoJoystick — библиотека для работы с джойстиком на Arduino Uno.  Считывает оси X/Y, отслеживает 6 кнопок + центральную, определяет 8 направлений + центр. Есть «мёртвая зона», поддержка INPUT_PULLUP, гибкая настройка пинов.  Совместима с Uno, Nano (ATmega328P). Лицензия MIT.  Простой API — быстрый старт!
